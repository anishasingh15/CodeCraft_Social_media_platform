\# 🌐 Social Media Web Application



A modern and responsive Social Media web application built with Django. The platform enables users to create accounts, share posts, interact with others, and manage their profiles through a clean and user-friendly interface.



\## 🚀 Features



\* User Registration \& Login

\* Secure Authentication

\* User Profile Management

\* Create, Edit, and Delete Posts

\* Feed to View Posts

\* Responsive User Interface

\* Django Admin Panel

\* SQLite Database Integration



\## 🛠️ Tech Stack



\* \*\*Frontend:\*\* HTML, CSS, JavaScript

\* \*\*Backend:\*\* Python, Django

\* \*\*Database:\*\* SQLite

\* \*\*Version Control:\*\* Git \& GitHub
